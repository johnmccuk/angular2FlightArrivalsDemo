Add a randomised Flight by clicking the button below or the icon on the Arrivals board.

So whats happening?

When you click the button, you access the FlightsComponent add method. This generates a random flight name and distance. It then passes these values to the FlightService addFlight method. It then adds an instance of the Flight class to the currentFlights array stored within the FlightService.

The constructor of each Flight class starts a timer, which subtracts the average speed of a 747 (520 mph) from the distance. The FlightsComponent basically just relays this information to the html view.
